Class JointPrismatic

A physical connection between two bodies with a defined axe movement. Used to create a sliding joint along one axis. Two RigidBodies need to be defined to use it. A motor can be defined to move the connected along the defined axis. Great to construct standard springs or physical sliders.

         JointHolder - bodyAnchor
                   ┌───┐
                   │   │
          <────────│   │──────> tied body, sliding on one Axis, 1 Degree of Freedom
                   │   │
                   └───┘
