Class JointRevolute

A physical connection between two bodies with a defined axe of rotation. Also known as HINGE joint. Two RigidBodies need to be defined to use it. A motor can be defined to rotate the connected along the defined axis.

                 rotation axis, 1st Degree of freedom
                   ↑
              ┌───┐│┌────┐
              │   │││    │
              │   │││    │
              │   │││    │
              └───┘│└────┘
                   │
     bodyAnchor         bodyTied
  (e.g. Doorhinge)       (e.g. Door)
